# 🚀 Installation

### Termux (Android)

```bash
termux-setup-storage
pkg update -y && pkg upgrade -y
pkg install git nodejs -y
git clone https://github.com/deveni0/Pomni-AI.git
cd Pomni-AI
npm install
npm start
```
